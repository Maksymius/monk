## Как тестировать

### Визуальное тестирование
1. Откройте обе версии в разных вкладках
2. Сравните внешний вид и анимации
3. Проверьте адаптивность на мобильных устройствах

### Функциональное тестирование
1. **Loader**: должен исчезнуть после загрузки
2. **Cursor glow**: должен следовать за курсором (только на десктопе)
3. **Parallax**: фоновые изображения должны загружаться
4. **Reveal animations**: элементы должны появляться при скролле
5. **Формы**: должны отправлять данные в Firebase

### Тестирование Firebase
1. Откройте http://localhost:8000/test-firebase.html
2. Нажмите "Test Firebase Connection"
3. Проверьте консоль браузера на наличие ошибок
4. Заполните формы на основном сайте
5. Проверьте Firebase Console на наличие новых записей

### Отладка
- Откройте Developer Tools (F12)
- Перейдите на вкладку Console
- Все операции Firebase логируются с префиксами ✅/❌
